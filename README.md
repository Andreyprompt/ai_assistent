## Описание проекта

Проект представляет собой интеллектуального ассистента, разработанного для автоматизации и оптимизации работы менеджеров по продажам в онлайн-университете **Zerocoder**. ИИ-ассистент способен эффективно взаимодействовать с клиентами, предоставляя точные ответы на вопросы, записывая на консультации и интегрируясь с инструментами бизнеса.

---

## Функционал ассистента

### 1. Осмысленное ведение диалога
Используя современные технологии обработки естественного языка (NLP), ИИ-ассистент поддерживает качественную коммуникацию с клиентами. Он способен понимать контекст запросов, задавать уточняющие вопросы и поддерживать логичный диалог.

### 2. Ответы на вопросы по базе знаний
Ассистент обучен использовать базу знаний университета **Zerocoder**, чтобы предоставлять актуальную информацию о курсах, программах обучения, форматах занятий и других деталях. Это позволяет минимизировать количество ручных обращений к менеджерам.

### 3. Запись клиента на консультацию
ИИ-ассистент помогает клиентам записаться на персональную консультацию с менеджером. Для этого он собирает необходимые данные (ФИО, контактные данные, удобное время) и автоматически создает встречу.

### 4. Интеграция с Google Календарь
После записи клиента система автоматически добавляет встречу в **Google Календарь** менеджера, что гарантирует отсутствие пропущенных консультаций и повышает организованность процесса.

---

## Используемые сервисы

### 1. **Qwen**
Сервис **Qwen** был использован для создания системного промпта и наполнения базы знаний ассистента. Благодаря гибкости платформы удалось выстроить четкую структуру диалогов и обеспечить высокое качество ответов.

### 2. **Suvvy (https://suvvy.ai/)**
Платформа **Suvvy** стала основой для разработки ИИ-ассистента. Она предоставляет мощные инструменты для создания ботов, работающих по сложным сценариям:
- **Точность ответов**: от 95%.
- **Обработка всех запросов**: Ассистент отвечает на 100% входящих сообщений.
- **Надежная база**: Платформа использует проверенные языковые модели (LLM).
- **Интеграции**: Возможность подключения к CRM, мессенджерам и маркетплейсам.

---

## Как это работает

1. **Клиент обращается к ассистенту** через сайт или мессенджеры, в нашем случае телеграмм
2. **ИИ анализирует запрос** и выбирает подходящий сценарий действий:
   - Если вопрос связан с базой знаний, ассистент предоставляет готовый ответ.
   - Если требуется запись на консультацию, ассистент собирает данные и создает встречу.
3. **Данные передаются в Google Календарь**, где менеджер может видеть все запланированные встречи.
4. **Менеджер проводит консультацию** с клиентом, имея всю необходимую информацию.

## Преимущества решения

- **Экономия времени менеджеров**: Рутинные задачи, такие как ответы на частые вопросы и запись на встречи, полностью автоматизированы.
- **Высокая точность**: Точность ответов составляет 95%, что минимизирует ошибки.
- **Удобство для клиентов**: Быстрая и качественная обратная связь повышает удовлетворенность клиентов.
- **Гибкость интеграций**: Возможность подключить дополнительные сервисы (CRM, мессенджеры, маркетплейсы).
